Cheat Sheets

Here is where I'll put all my notes

  • Cheat 1: To add spinning picture, add in trasition: into the .picture and add rotate: to .picture:hover
  • Cheat 2: chrome.storage.sync.get will help you store and use input from users on chrome.
  • Cheat 3: In p5, if you want an image to move as a whole, translate the whole center point in the very start.
  • Cheat 4: A treeshake command in the package.json helps with getting the p5.play to work on gh pages
  • Cheat 5: Some of the p5.play documentation is very outdated, use other more updated resources for it
  • Cheat 6: Rapid API is an amazing resource for all things API, guides you through the set up and will write code pieces for you.
